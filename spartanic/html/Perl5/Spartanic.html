NAME

Perl5::Spartanic - By using of this package you declare your module to be spartanic.

VERSION

This documentation refers to version 0.101 of Perl5::Spartanic

SYNOPSIS

  use Perl5::Spartanic;

DESCRIPTION

By using of this package you declare your module to be spartanic, that means:

It is a Pure-Perl module

  • No compile

  • No install

  • Just copy file

Uses only

  • Core modules

  • Some other spartanic modules
